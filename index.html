Como Usar

Guia rápido para usar cada aba do ImobCore.

1) Proprietários

Cadastre os donos dos imóveis com nome e contato.

2) Locatários

Cadastre os inquilinos que vão assinar contratos.

3) Imóveis

Cadastre cada imóvel, vinculando ao proprietário.

4) Contratos

Crie contratos com datas, vencimento e valores mensais.

5) Dados do Recibo

Preencha seus dados para sair automaticamente no recibo impresso.

6) Recibos

Gere, visualize, imprima e salve o histórico mensal de recibos.

7) Financeiro

Registre baixas do locatário, repasses ao proprietário e gere extratos demonstrativos.

8) Relatórios

Filtre por tipo, status, mês e pagamento para análise de carteira.

9) Dashboard

Veja indicadores e vencimentos próximos de forma rápida.

Atualização

Verifique e instale novas versões do ImobCore.

Sobre

Informações do sistema ImobCore.

Sobre o Sistema

ImobCore é um sistema desktop para controle de imóveis, contratos de aluguel, recibos e relatórios gerenciais.

Funcionalidades

Cadastro de proprietários, locatários e imóveis, gestão de contratos, geração de recibos, painel de vencimentos e relatórios para impressão.
